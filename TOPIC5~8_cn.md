## 🧬 TOPIC 5.1 — Meiosis（减数分裂）

### LO 5.1.A：解释减数分裂如何导致染色体从一代传递到下一代

**【核心定义】**
减数分裂是**有性生殖的二倍体生物**中确保形成**单倍体（haploid）配子细胞**（有时称为子细胞）的过程。配子的染色体数目是体细胞的一半（n vs 2n）。

---

**【减数分裂 I —— 详细分步】**

**① 前期 I（Prophase I）—— 最复杂、最关键的一步**

- **同源染色体配对并凝缩**：来自父方和母方的两条同源染色体（homologous chromosomes）互相靠近并螺旋化变短变粗
- **联会（Synapsis）**：同源染色体紧密配对，形成**二价体（bivalent）**，即**四分体（tetrad）**——4条染色单体
- **交叉（Chiasmata）形成**：在联会区域，**非姐妹染色单体**之间可能发生物理交换点
  - ⚠️ 交叉是**交叉互换（crossing over）**的结构基础，由此产生遗传重组（详见5.2）
- **减数分裂纺锤体开始形成**
- **中心体（centrosomes）移向细胞两极**
- **核膜（nuclear envelope）解体**

> 💡 **与有丝分裂的关键差异**：有丝分裂前期没有同源染色体配对和联会。

**② 中期 I（Metaphase I）**

- 纺锤丝（微管）附着在**动粒（kinetochore）**上
- **同源染色体对**（而不是单个染色体）排列在细胞**赤道板（metaphase plate）**上
- ⚠️ **关键区别**：每条染色体的一侧动粒连接到一极的纺锤丝，而其同源染色体的相同侧动粒连接到**另一极**的纺锤丝
- **同源染色体的排列方向是随机的**——这就是**随机分配（independent assortment）**的基础

**③ 后期 I（Anaphase I）**

- **同源染色体彼此分离**，被纺锤丝拉向两极
- ⚠️ 但**姐妹染色单体（sister chromatids）仍然连接在着丝粒上**——这是与有丝分裂后期最关键的差异
- 染色体数目由此**减半**（2n → 1n 分配到两极）

**④ 末期 I（Telophase I）**

- 纺锤体解体
- 核膜重新形成（染色体可能在两极暂时解凝缩）
- **胞质分裂（Cytokinesis）**：
  - **动物细胞** → **分裂沟（cleavage furrow）**形成
  - **植物细胞** → **细胞板（cell plate）**形成
- 结果：形成 **2 个单倍体子细胞**（每个含复制的染色体，即每条染色体仍有两条姐妹染色单体）

---

**【减数分裂 II —— 详细分步】**

① **前期 II（Prophase II）**
- 纺锤体重新形成
- 姐妹染色单体通过**着丝粒（centromere）**连接到纺锤体

② **中期 II（Metaphase II）**
- 染色体排列在赤道板上
- 每个染色单体的**动粒**都附着在一极的微管上
- ⚠️ 与有丝分裂中期几乎相同

③ **后期 II（Anaphase II）**
- **着丝粒蛋白解体**
- **姐妹染色单体分开**，被拉向相反两极
- ⚠️ 此时每条染色单体才真正成为**独立的染色体**

④ **末期 II（Telophase II）**
- 纺锤体解体 → 核膜重新形成 → 染色质解凝缩 → 胞质分裂
- 结果：形成 **4 个单倍体子细胞**，每个含**未复制的染色单体（即一条染色体）**

**最终产物**：4个遗传上各不相同的单倍体配子（如精子或卵细胞）。

---

### LO 5.1.B：描述有丝分裂与减数分裂的阶段和结果的异同

| 比较维度 | 有丝分裂（Mitosis） | 减数分裂（Meiosis） |
|----------|-------------------|-------------------|
| **发生场景** | 体细胞生长、修复、无性繁殖 | 生殖细胞产生配子 |
| **分裂次数** | 1次 | 2次（I + II） |
| **子细胞数** | 2个 | 4个 |
| **子细胞染色体数** | 2n（二倍体，与亲代相同） | 1n（单倍体，亲代的一半） |
| **子细胞遗传组成** | 与亲代完全相同（克隆） | 各不相同（重组） |
| **前期是否配对** | 否 | **是（联会 + 交叉）** |
| **有丝分裂/减I后期** | 姐妹染色单体分离 | **同源染色体分离**（姐妹染色单体不分离） |
| **功能** | 生长、组织修复、无性生殖 | 有性生殖中的配子形成 |
| **共同点** | 都使用纺锤体装置移动染色体 | 都使用纺锤体装置移动染色体 |

## 🧬 TOPIC 5.2 — Meiosis and Genetic Diversity（减数分裂与遗传多样性）

### LO 5.2.A：解释减数分裂过程如何产生遗传多样性

**【三大遗传多样性来源】**

**① 交叉互换（Crossing Over）/ 重组（Recombination）**
- 时间：**减数分裂 I 的前期 I**
- 位置：**非姐妹染色单体**之间（即来自不同亲本的两条染色单体）
- 机制：在交叉（chiasmata）处，非姐妹染色单体交换相应片段的DNA
- 效果：产生新的等位基因组合 → **增加配子间的遗传多样性**

**② 随机分配（Independent Assortment）**
- 时间：**减数分裂 I 的中期 I**
- 机制：同源染色体对在赤道板上的排列方向是随机的
  - 母方染色体可以朝任何一极，父方染色体随之相对排列
- 计算公式：对于 n 对染色体，可能的组合方式 = **2ⁿ**
- 人类（23对染色体）：$2^{23} \approx 8.4 \times 10^6$ 种组合
- ⚠️ 自由组合定律适用于**位于不同染色体上**的基因

**③ 受精作用（Fertilization）**
- 两个单倍体配子融合，恢复二倍体
- 随机选择精子（约2亿个中选1个）+ 随机选择卵子 → 组合爆炸
- 人类可能产生的**合子基因型组合** ≈ $2^{23} \times 2^{23} \approx 7 \times 10^{13}$ 种

---

**【非整倍体（Nondisjunction）—— 分离错误】**

- 当**同源染色体在减数 I** 或**姐妹染色单体在减数 II** 未能正常分离时发生
- 结果：某些配子获得额外染色体（n+1），某些缺少染色体（n-1）
- 受精后可能导致**非整倍体（aneuploidy）**，如三体（trisomy）或单体（monosomy）

> ❌ **Exclusion**：各种动植物有性生殖周期的细节不考。

## 🧬 TOPIC 5.3 — Mendelian Genetics（孟德尔遗传）

### LO 5.3.A：用孟德尔定律解释基因和性状的遗传

**【孟德尔第一定律：分离定律（Law of Segregation）】**
- 每个个体携带一对等位基因（alleles），在形成配子时**彼此分离**
- 每个配子**只**携带一对等位基因中的一个
- 基础：**减数分裂 I 同源染色体的分离**

**【孟德尔第二定律：自由组合定律（Law of Independent Assortment）】**
- 位于**不同染色体上**的基因在形成配子时彼此独立分配
- 基础：**减数分裂 I 中期同源染色体的随机排列**
- ⚠️ 位于同一染色体上的**连锁基因（linked genes）**不遵循自由组合定律

---

**【核心术语精确定义】**

| 术语 | 定义 | 备注 |
|------|------|------|
| **基因型（Genotype）** | 个体对一个或多个基因继承的等位基因集合 | 可纯合或杂合 |
| **表型（Phenotype）** | 继承性状的可观察表达 | 是基因型与环境相互作用的结果 |
| **纯合（Homozygous）** | 两个等位基因相同（如 AA 或 aa） | |
| **杂合（Heterozygous）** | 两个等位基因不同（如 Aa） | |
| **显性（Dominant）** | 杂合时仍能表达的等位基因 | 通常用大写字母表示 |
| **隐性（Recessive）** | 杂合时不表达，纯合时才表达 | 通常用小写字母表示 |

**【杂交分析工具】**

1. **单因子杂交（Monohybrid Cross）**：追踪一对相对性状 → 表型比 3:1（F2代）
2. **双因子杂交（Dihybrid Cross）**：追踪两对相对性状 → 表型比 9:3:3:1（F2代，假设自由组合）
3. **测交（Test Cross）**：将未知基因型的个体与**隐性纯合子**杂交 → 根据子代表型推断亲代基因型
4. **Punnett 方格**：预测亲代和子代的基因型和表型
5. **系谱分析（Pedigree Analysis）**：通过家族系谱推断遗传模式（常染色体显性/隐性、伴X显性/隐性、伴Y）

**【概率规则（考纲明确要求）】**

| 情形 | 规则 | 公式 |
|------|------|------|
| **互斥事件** | 事件A或事件B发生的概率 | $P(A \text{ or } B) = P(A) + P(B)$ |
| **独立事件** | 事件A和事件B同时发生的概率 | $P(A \text{ and } B) = P(A) \times P(B)$ |

## 🧬 TOPIC 5.4 — Non-Mendelian Genetics（非孟德尔遗传）

### LO 5.4.A：解释偏离孟德尔遗传模型的情况

**【6种偏离类型 详细展开】**

---

**① 基因连锁（Genetic Linkage）**

- **定义**：位于同一染色体上的基因在减数分裂时倾向于一起遗传
- **后果**：不遵循自由组合定律 → 子代中**亲代组合（parental types）**远多于**重组组合（recombinant types）**
- **遗传图谱构建**：
  - 重组频率（recombination frequency）用于计算基因在染色体上的**相对距离**
  - **1% 重组频率 = 1 图谱单位（map unit） = 1 厘摩（cM）**
  - 公式：$重组频率 = \frac{重组子代数量}{总子代数量} \times 100\%$
  - 两点之间重组频率越低 → 距离越近 → 连锁越紧密

---

**② 共显性（Codominance）**

- **定义**：两个等位基因在杂合子中**同时、独立表达**
- 杂合子的表型**不是**中间类型，而是**两个纯合表型的叠加**
- **经典例子**：
  - ABO 血型系统的 **AB 型**（$I^AI^B$）→ 既有 A 抗原又有 B 抗原
  - 斑马鱼某些颜色基因
  - 牛的毛色（红白斑驳）

---

**③ 不完全显性（Incomplete Dominance）**

- **定义**：没有等位基因能完全掩盖另一个
- 杂合子的表型是两个纯合表型的**中间类型**（"混合"）
- **经典例子**：
  - 金鱼草（snapdragon）：红花（RR）× 白花（rr）→ 粉花（Rr）
  - 人的卷发类型：卷发（CC）× 直发（cc）→ 波浪发（Cc）

> ⚠️ **共显性 vs 不完全显性的区别**：共显性是两者同时出现（如AB血型），不完全显性是两者混合成中间状态（如粉色）。

---

**④ 伴性遗传（Sex-Linked Inheritance）**

- 性状由**性染色体**（X 或 Y）上的基因决定
- **X 连锁隐性遗传**的关键特征：
  - **XY 个体（男性）** 只有一条 X 染色体 → 只要继承一个隐性等位基因就会患病
  - **XX 个体（女性）** 需要两个隐性等位基因才会患病
  - 因此：**XY 个体的发病率远高于 XX 个体**
  - 父方不能将 X 连锁性状传给儿子（儿子从父方得到Y）
- **经典例子**：红绿色盲、血友病A、Duchenne 肌营养不良
- **非 X/Y 性别决定系统**（考纲举例）：
  - 鸟类：**ZW 系统**（ZZ = 雄性，ZW = 雌性）
  - 蜜蜂：**染色体倍性**（二倍体 = 雌蜂，单倍体 = 雄蜂）

---

**⑤ 多效性（Pleiotropy）**

- **定义**：**一个基因**的表达影响**多个看似无关的表型性状**
- 这些表型性状因此**不独立分离**
- **经典例子**：
  - **镰状细胞病（Sickle Cell Disease）**：
    - HBB 基因的一个突变 → 血红蛋白结构异常
    - 导致：红细胞镰状变形、贫血、疼痛危象、脾脏损伤、对疟疾的部分抵抗力
  - **马凡综合征（Marfan Syndrome）**：
    - FBN1 基因突变 → 结缔组织异常
    - 导致：身高异常高、四肢细长、晶状体脱位、主动脉瘤风险高
  - **苯丙酮尿症（PKU）**：PAH 基因突变 → 多个神经系统发育异常

---

**⑥ 非核遗传（Non-nuclear Inheritance）/ 细胞质遗传**

- 线粒体和叶绿体含有自己的DNA，编码部分蛋白质
- **关键特征**：
  - 这些细胞器**随机分配**到配子和子细胞中
  - 不遵循孟德尔的分离和自由组合定律
  - 表现为**母系遗传（Maternal Inheritance）**

| 生物类型 | 传递方式 | 线粒体来自 | 叶绿体来自 |
|----------|---------|-----------|-----------|
| **动物** | 线粒体通过卵子传递，**不通过精子** | **母方** | N/A |
| **植物** | 线粒体和叶绿体在**胚珠（ovule）**中传递，**不在花粉（pollen）**中 | **母方** | **母方** |

- **经典例子**：
  - 人类线粒体疾病（Leber's遗传性视神经病变，MELAS综合征）
  - 紫茉莉（Mirabilis jalapa）的叶色遗传

## 🧬 TOPIC 5.5 — Environmental Effects on Phenotype（环境对表型的影响）

### LO 5.5.A：解释相同基因型在不同环境条件下如何产生多种表型

**【表型可塑性（Phenotypic Plasticity）】**

- **定义**：同一个基因型在不同环境条件下产生不同表型的能力
- **机制**：环境条件**影响基因表达**（通过调控转录、翻译、蛋白质修饰等）
- 表型 = **基因型 + 环境**

**💡 考纲提供的经典示例**：

| 示例 | 解释 |
|------|------|
| **人类身高和体重** | 受营养、运动等环境因素显著影响 |
| **花的颜色随土壤pH变化** | 如绣球花（Hydrangea）：酸性土壤→蓝色花，碱性土壤→粉色花 |
| **北极动物季节性毛色** | 如北极狐：冬季白色（雪地伪装），夏季棕色或灰色 |
| **爬行动物的性别决定** | 如海龟、鳄鱼：**温度依赖性性别决定（TSD）**——孵化温度决定幼体性别 |
| **紫外线对黑色素的影响** | 紫外线照射增加动物（包括人类）的黑色素生成（晒黑） |
| **真菌信息素产生** | 酵母和其他真菌中，相反交配型的存在影响信息素的产生 |

## 🧬 TOPIC 6.1 — DNA and RNA Structure（DNA与RNA结构）

### LO 6.1.A：描述参与将遗传信息从一代传递到下一代的结构

**【遗传物质的化学形式】**
- 遗传信息存储在 **DNA 分子**中；某些病毒以 **RNA** 作为遗传物质

**【染色体结构对比】**

| 特征 | 原核生物 | 真核生物 |
|------|---------|---------|
| 染色体形态 | **环状（circular）**，通常只有一条 | **多条线性（linear）** |
| 染色体位置 | 拟核（nucleoid），无核膜包裹 | 细胞核内，有核膜包裹 |
| DNA压缩 | 无组蛋白（histones），但有类组蛋白 | 与**组蛋白**和相关蛋白质结合形成**染色质（chromatin）** |
| 额外DNA | 可有**质粒（plasmids）**——染色体外的小环状DNA | 线粒体和叶绿体有自身环状DNA |

---

### LO 6.1.B：描述DNA作为遗传材料的特性

**【核苷酸结构】**
每个核苷酸由三部分组成：
1. **磷酸基团**（Phosphate group）
2. **五碳糖**：DNA = 脱氧核糖（deoxyribose），RNA = 核糖（ribose）
3. **含氮碱基**（Nitrogenous base）

**【嘌呤（Purines）vs 嘧啶（Pyrimidines）】**

| 类别 | 碱基 | 环结构 | 配对规则 |
|------|------|--------|---------|
| **嘌呤（Purines）** | 腺嘌呤（**A**）、鸟嘌呤（**G**） | **双环**结构 | 嘌呤配对嘧啶 |
| **嘧啶（Pyrimidines）** | 胞嘧啶（**C**）、胸腺嘧啶（**T**，仅DNA）、尿嘧啶（**U**，仅RNA） | **单环**结构 | 嘧啶配对嘌呤 |

**【碱基配对规则（Chargaff's Rules）】**

| 分子 | 配对关系 | 氢键数 |
|------|---------|--------|
| **DNA** | A = T；G ≡ C | A-T：2个氢键；G-C：3个氢键 |
| **RNA** | A = U；G ≡ C | A-U：2个氢键；G-C：3个氢键 |

- 碱基配对在进化中是**高度保守**的
- DNA 双链是**反向平行（antiparallel）**的：一条链 5'→3'，另一条 3'→5'

## 🧬 TOPIC 6.2 — DNA Replication（DNA复制）

### LO 6.2.A：描述遗传信息在代际间复制的机制

**【DNA复制的基本原则】**

1. **方向性**：DNA 聚合酶只能以 **5' → 3'** 方向合成新链
2. **半保留复制（Semiconservative Replication）**：
   - 亲代DNA双链解开，每条亲代链作为合成新互补链的模板
   - 每个子代DNA分子包含**一条亲代链**和**一条新合成的链**
   - 这是 Meselson-Stahl 实验（1958）证明的

**【DNA复制的详细步骤】**

| 步骤 | 参与者 | 功能 |
|------|--------|------|
| **① 解旋** | **解旋酶（Helicase）** | 解开DNA双链，形成**复制叉（replication fork）** |
| **② 解超螺旋** | **拓扑异构酶（Topoisomerase）** | 在复制叉前方**松弛超螺旋**，防止DNA打结 |
| **③ 提供起始点** | **引物酶（Primase，一种RNA聚合酶）** | 合成**RNA引物（RNA primer）**，提供DNA聚合酶起始所需的3'-OH末端 |
| **④ 合成新链** | **DNA聚合酶（DNA Polymerase）** | 读取模板链，添加互补核苷酸 |
| **⑤ 连接片段** | **连接酶（Ligase）** | 在**后随链**上连接**冈崎片段（Okazaki fragments）** |

**【前导链 vs 后随链】**

| 特征 | 前导链（Leading Strand） | 后随链（Lagging Strand） |
|------|------------------------|------------------------|
| 合成方向 | 连续向复制叉方向 | **不连续**，远离复制叉方向 |
| 所需引物数 | 1 个（起始时） | 多个（每个冈崎片段需要一个） |
| 产物 | 连续的长DNA链 | 一系列**冈崎片段** |
| 连接 | 不需要 | 需要**连接酶**连接片段 |

> ❌ **Exclusion**：除 DNA 聚合酶、连接酶、RNA 聚合酶、解旋酶和拓扑异构酶外的其他酶的名称和步骤细节不考。

## 🧬 TOPIC 6.3 — Transcription and RNA Processing（转录与RNA加工）

### LO 6.3.A：描述遗传信息从DNA流向RNA再到蛋白质的机制

**【三种RNA的详细功能】**

| RNA类型 | 中文名称 | 功能 | 位置 |
|---------|---------|------|------|
| **mRNA** | 信使RNA | 携带DNA的遗传信息从细胞核到细胞质核糖体 | 核内合成 → 进入细胞质 |
| **tRNA** | 转运RNA | 携带特定氨基酸到核糖体；具有**反密码子**与mRNA密码子配对 | 细胞质 |
| **rRNA** | 核糖体RNA | 核糖体的**功能性构建模块**（催化肽键形成） | 核糖体 |

---

**【转录过程详解】**

1. **起始**：RNA聚合酶结合到DNA的**启动子区域（promoter）**，解开局部双链
2. **延伸**：
   - RNA聚合酶读取 **DNA模板链 3'→5'**
   - 以 **5'→3'** 方向合成RNA分子
   - 碱基配对规则：A→U，T→A，C→G，G→C
3. **终止**：遇到**终止序列**，RNA聚合酶释放新合成的RNA分子

**⚠️ 关键区分**：
- 模板链（Template strand / Antisense strand）：被 RNA 聚合酶读取的链
- 编码链（Coding strand / Sense strand）：与RNA序列相同（T→U）

---

**【真核生物RNA加工（3个主要修饰）】**

**① 5'端加帽（5' Capping）**
- 在mRNA的5'端添加一个修饰的**鸟嘌呤核苷酸（GTP cap）**
- 功能：帮助核糖体识别mRNA、防止被核酸外切酶降解

**② 3'端加尾（Poly-A Tail）**
- 在mRNA的3'端添加**50-250个腺嘌呤核苷酸**
- 功能：增加mRNA稳定性、促进核输出、帮助翻译起始

**③ RNA剪接（RNA Splicing）**

| 术语 | 定义 |
|------|------|
| **内含子（Introns）** | **非编码**序列，被**切除** |
| **外显子（Exons）** | **编码**序列，被**保留**并连接 |

- **选择性剪接（Alternative Splicing）**：
  - 同一个前体mRNA可以通过不同的剪接方式，产生**多个不同的成熟mRNA版本**
  - 一个基因 → 多种蛋白质异构体（protein isoforms）
  - 极大地增加了蛋白质组的多样性而不增加基因数量
  - ⚠️ 这是考纲特别强调的概念！

---

## 🧬 TOPIC 6.4 — Translation（翻译）

### LO 6.4.A：解释生物体的表型如何由其基因型决定

**【翻译的场所】**
- 翻译在**核糖体**上进行
- **原核生物**：核糖体在细胞质中
- **真核生物**：核糖体在细胞质中或**粗面内质网（rough ER）**表面

**【原核生物的特殊性——耦合转录-翻译】**
- 在原核生物中，**转录尚未完成时，翻译就已经开始**
- mRNA仍与DNA结合时，核糖体即可结合到mRNA的5'端开始翻译
- 这是因为原核生物没有核膜分隔转录和翻译

---

**【翻译的详细三步流程】**

**① 起始（Initiation）**
1. 核糖体小亚基结合到mRNA
2. 在**起始密码子 AUG** 处（编码**甲硫氨酸 methionine**）识别起始位点
3. 携带甲硫氨酸的起始tRNA（甲硫氨酸-tRNA）结合到AUG
4. 核糖体大亚基结合，形成完整核糖体

**② 延伸（Elongation）**
1. **密码子识别**：mRNA上的**三联体密码子**被核糖体读取
2. **tRNA递送**：携带对应反密码子的tRNA将正确氨基酸带入A位点
3. **肽键形成**：核糖体催化肽键形成，多肽链延长
4. **移位（Translocation）**：核糖体沿mRNA移动3个核苷酸（一个密码子）
5. 重复上述步骤

**③ 终止（Termination）**
1. 核糖体遇到**终止密码子（stop codon）**：UAA、UAG 或 UGA
2. 没有tRNA识别终止密码子；**释放因子（release factor）** 结合
3. 多肽链从tRNA上释放
4. 核糖体解离，新合成的蛋白质被释放

---

**【遗传密码的关键特征】**

- **简并性（Degeneracy）**：大多数氨基酸由**多个密码子**编码（如亮氨酸Leu有6个密码子）
- **通用性（Universal）**：几乎所有生物使用**相同的遗传密码** → 这是**共同祖先**的有力证据！⭐

> ❌ **Exclusion**：遗传密码表不需要背诵（除了 AUG 起始密码子）。

**【逆转录病毒（Retroviruses）—— 特殊的信息流】**
- 正常信息流：DNA → RNA → 蛋白质
- 逆转录病毒信息流：**RNA → DNA → RNA → 蛋白质**
- 酶：**逆转录酶（Reverse Transcriptase）**
  - 将病毒RNA基因组逆转录为DNA
  - 该DNA**整合到宿主基因组中**
  - 利用宿主细胞的转录翻译系统产生新的病毒后代
- **经典例子**：HIV（人类免疫缺陷病毒）

## 🧬 TOPIC 6.5 — Regulation of Gene Expression（基因表达调控）

### LO 6.5.A：描述调控基因表达的相互作用类型

**【调控的多个层面】**

| 层面 | 原核生物 | 真核生物 |
|------|---------|---------|
| **转录水平** | 操纵子系统（主要） | 转录因子、增强子、沉默子、染色质重塑 |
| **转录后水平** | 较少 | RNA剪接、RNA编辑、RNA稳定性 |
| **翻译水平** | 较少 | 翻译调控因子、mRNA定位 |
| **翻译后水平** | 较少 | 蛋白质修饰、降解 |

---

**【关键概念】**

1. **调控序列（Regulatory Sequences）**：一段段与调控蛋白相互作用的DNA序列，控制转录是否发生
2. **组成型表达（Constitutive Expression）**：某些基因在所有细胞中持续表达（如"持家基因"housekeeping genes）
3. **可诱导表达（Inducible Expression）**：某些基因只有在需要时才被开启

---

**【表观遗传调控（Epigenetic Regulation）】**

- **定义**：不改变DNA序列的**可逆修饰**，影响基因表达
- **两种主要机制**：

| 机制 | 描述 | 效果 |
|------|------|------|
| **DNA甲基化（DNA Methylation）** | 在胞嘧啶（C）上添加甲基（-CH₃）基团 | 通常**抑制**基因表达（关闭基因） |
| **组蛋白修饰（Histone Modification）** | 组蛋白上的乙酰化、甲基化、磷酸化等 | 乙酰化→**激活**；去乙酰化→**抑制** |
| **染色质重塑（Chromatin Remodeling）** | 改变DNA与组蛋白的缠绕密度 | 松散→可转录；紧密→不可转录 |

---

**【基因表达与表型的关系（考纲核心观点）】**

1. **细胞分化（Cell Differentiation）**：所有体细胞含相同DNA，但**差异基因表达**导致细胞特化
2. **组织特异性蛋白**：不同细胞类型表达不同组合的基因 → 产生不同的蛋白质 → 实现不同功能
3. **发育过程中的顺序性表达**：转录因子诱导 → 启动下游基因的级联表达
4. **基因产物的功能和数量决定表型**

---

### LO 6.5.B：解释调控序列位置与其功能的关系

**【原核生物：操纵子系统（Operon System）】**

- **操纵子** = 一组在转录水平上协同调控的基因
- 结构：
  - **启动子（Promoter）**：RNA聚合酶结合位点
  - **操纵基因（Operator）**：调控蛋白结合位点
  - **结构基因（Structural genes）**：被协同表达的编码基因
- **诱导型系统（Inducible）**：如 **lac 操纵子**
  - 通常关闭，存在乳糖时开启
- **抑制型系统（Repressible）**：如 **trp 操纵子**
  - 通常开启，存在色氨酸时关闭

> ⚠️ 考纲特别提示：学生可以用 lac 操纵子帮助理解**正/负基因调控**

**【真核生物：协同调控的机制】**
- 多个基因可能受**相同的转录因子**调控
- 这些基因不一定在染色体上相邻
- 例如：响应同一激素的多个基因

---

## 🧬 TOPIC 6.6 — Gene Expression and Cell Specialization（基因表达与细胞特化）

### LO 6.6.A：解释转录因子与启动子区的结合如何影响基因表达和表型

**【转录起始的分子机制】**

1. **正调控（Positive Regulation）**：
   - **转录因子（Transcription Factors）** 结合到**启动子**或**增强子（enhancer）** 序列
   - 增强子可以位于基因的**上游或下游**，甚至距离基因数千碱基
   - 通过DNA环化（looping），增强子与启动子接触 → 激活转录
   - 招募RNA聚合酶 → 启动转录

2. **负调控（Negative Regulation）**：
   - **阻遏蛋白（Repressor Proteins）** 结合到**沉默子（silencer）** 或操纵基因
   - 阻止RNA聚合酶结合或延伸 → **抑制转录**
   - 例：lac 操纵子的 lac 阻遏蛋白

---

### LO 6.6.B：解释基因表达调控与细胞和生物体表型差异之间的联系

- **差异基因表达** → 同一生物的不同细胞产生不同的蛋白质 → 实现不同功能
- 例如：神经细胞表达神经递质相关基因，肌肉细胞表达肌动蛋白/肌球蛋白基因
- **小RNA的调控作用**（考纲明确要求）：
  - **miRNA（microRNA）**：~22nt 小RNA，与靶mRNA不完全互补 → 抑制翻译或促进mRNA降解
  - **siRNA（small interfering RNA）**：~20-25nt，与靶mRNA完全互补 → 导致mRNA降解（RNA干扰）

## 🧬 TOPIC 6.7 — Mutations（突变）

### LO 6.7.A：描述各种类型的突变

**【按分子变化分类的突变类型】**

| 突变类型 | 定义 | 是否改变阅读框 | 对蛋白质的影响 |
|----------|------|--------------|--------------|
| **点突变（Point Mutation）** | 一个核苷酸被另一个替换 | **不改变** | 取决于具体替换 |
| —— **同义/沉默突变（Silent）** | 核苷酸改变，但编码相同氨基酸 | 不改变 | **无变化** |
| —— **错义突变（Missense）** | 核苷酸改变，编码不同氨基酸 | 不改变 | 可能改变蛋白质结构/功能 |
| —— **无义突变（Nonsense）** | 变为终止密码子，**提前终止** | 不改变阅读框但截断蛋白质 | **通常丧失功能** |
| **移码突变（Frameshift）** | 一个或多个核苷酸的插入或缺失 | **改变阅读框**⭐ | 通常产生完全不同的、无功能的蛋白质 |

> ⚠️ **考纲特别强调的常见误区**：
> 1. 学生常误以为所有突变都是有害的——**错！** 突变可以是有益的、有害的或中性的
> 2. 学生常混淆"突变导致蛋白质变性"——**错！** 变性是指蛋白质构象改变，不是突变导致的
> 3. **沉默突变**的存在证明不是所有点突变都导致蛋白质变化

---

**【突变结果的3种可能性】**

| 结果 | 描述 | 示例 |
|------|------|------|
| **有益突变（Beneficial）** | 增加生物在特定环境中的适应度 | MC1R基因突变→口袋鼠的适应性黑化 |
| **有害突变（Detrimental）** | 降低生物适应度 | CFTR基因突变→囊性纤维化 |
| **中性突变（Neutral）** | 对适应度无显著影响 | 同义突变（silent mutation） |

**💡 考纲举例**：
- CFTR基因突变 → 破坏氯离子运输 → **囊性纤维化（cystic fibrosis）**
- MC1R基因突变 → **口袋鼠的适应性黑化（adaptive melanism）**
- 镰状细胞贫血（既是疾病也是杂合子优势）

> ❌ **Exclusion**：特定突变及其效应的具体知识不考（上述仅为说明性示例）。

---

### LO 6.7.B：解释基因型的改变如何导致表型的改变

**【突变的来源（3种）】**

1. **DNA复制错误**：DNA聚合酶偶尔会接入错误核苷酸
   - 细胞有**错配修复（mismatch repair）**机制，但无法100%修复
2. **DNA修复机制缺陷**：修复系统的基因本身发生突变 → 修复能力下降
3. **外部因素（诱变剂 mutagens）**：
   - **物理因素**：电离辐射（X射线、γ射线）、紫外线（形成嘧啶二聚体）
   - **化学因素**：活性化学物质、亚硝酸盐、黄曲霉素等

---

**【染色体数目异常——非整倍体（Aneuploidy）】**

| 类型 | 定义 | 产生机制 |
|------|------|---------|
| **三体（Trisomy）** | 某条染色体有3个拷贝（2n+1） | 减数分裂**不分离（nondisjunction）** |
| **单体（Monosomy）** | 某条染色体只有1个拷贝（2n-1） | 减数分裂不分离 |

- 非整倍体通常导致**发育受限的疾病**
- 在人类中，大多数非整倍体是致死性的（早期流产）
- 少数可存活：**唐氏综合征（21三体）**、特纳综合征（XO）、克兰费尔特综合征（XXY）
- 不分离可发生在**减数分裂 I**（同源染色体不分）或**减数分裂 II**（姐妹染色单体不分）

> ❌ **Exclusion**：与染色体数目改变相关的特定疾病不考。

---

### LO 6.7.C：解释DNA序列的改变如何产生可被自然选择作用的变异

**【突变是进化的原料】**
- 突变是**遗传变异的根本来源**
- 没有突变 → 没有新等位基因 → 自然选择没有材料 → 进化停滞

**【原核生物增加遗传变异的4种机制】**

| 机制 | 英文 | 描述 | 方向 |
|------|------|------|------|
| **转化** | Transformation | 细胞从环境中**吸收游离DNA**（如死亡细菌释放的DNA） | **水平基因转移** |
| **转导** | Transduction | **病毒（噬菌体）** 将DNA从一个细菌传递到另一个细菌 | **水平基因转移** |
| **接合** | Conjugation | 两个细菌通过**性菌毛（pilus）**直接接触，转移质粒或DNA片段 | **水平基因转移** |
| **转座** | Transposition | DNA片段（转座子/跳跃基因）在基因组内或基因组间**移动位置** | 可导致基因重排 |

- **病毒重组**：相关病毒感染同一宿主细胞时可以**重组遗传信息**
- 这些增加遗传变异的机制在进化上**高度保守**，被多种生物共享

## 🧬 TOPIC 6.8 — Biotechnology（生物技术）

### LO 6.8.A：解释基因工程技术在分析或操作DNA中的用途

**【四大核心技术】**

---

**① 凝胶电泳（Gel Electrophoresis）**

| 参数 | 原理 |
|------|------|
| **分离依据** | DNA片段**大小**和**电荷** |
| **基本步骤** | 将DNA样品点入琼脂糖凝胶的孔中→通电→DNA（带负电）向正极移动→较小片段移动更快更远 |
| **结果** | 形成**DNA条带（DNA fingerprint）** |
| **应用** | 法医学鉴定、亲子鉴定、物种鉴定、系统发育分析 |

---

**② 聚合酶链式反应（PCR, Polymerase Chain Reaction）**

- 目的：**指数级扩增**特定DNA片段
- **3个步骤的循环（重复25-35次）**：

| 步骤 | 温度 | 过程 |
|------|------|------|
| **变性（Denaturation）** | ~95°C | 双链DNA解链为单链 |
| **退火（Annealing）** | ~50-65°C | 引物（primers）与目标DNA序列特异性结合 |
| **延伸（Extension）** | ~72°C | DNA聚合酶（Taq polymerase）从引物开始合成新链 |

- 每个循环使目标DNA量**翻倍**

---

**③ 细菌转化（Bacterial Transformation）**

- 将**外源DNA**（如重组质粒）导入细菌细胞
- 常用的细菌：**大肠杆菌（E. coli）**
- 标记基因：通常包含**抗生素抗性基因**（如amp^R）以便筛选成功转化的细菌
- 应用：
  - **基因克隆**：将外源DNA片段插入质粒→转化细菌→大量复制
  - **生产重组蛋白**：如人胰岛素、生长激素
  - **转基因生物（GMOs）**：将外源基因导入动植物基因组

---

**④ DNA测序（DNA Sequencing）**

- 确定DNA分子中核苷酸的**精确排列顺序**
- 现代高通量测序技术可以快速读取大量序列
- 应用：
  - 比较不同物种的DNA序列 → **系统发育分析**
  - 法医学鉴定
  - 诊断遗传疾病
  - 基因组学研究

> ❌ **Exclusion**：每种基因工程技术的具体操作细节不考。你需要理解**原理和用途**。

## 🌿 TOPIC 7.1 — Introduction to Natural Selection（自然选择导论）

### LO 7.1.A：描述自然选择的原因

**【达尔文自然选择理论的核心逻辑链】**

```
过度繁殖（Overproduction）
      ↓
有限资源 → 生存竞争（Struggle for Existence）
      ↓
个体间存在变异（Variation）
      ↓
有利变异 → 生存率更高 → 更多后代（Differential Survival & Reproduction）
      ↓
有利性状代代相传（Descent with Modification）
      ↓
种群遗传组成随时间改变 → 进化（Evolution）
```

**【关键表述（考试注意）】**

- **自然选择是进化的主要机制（major mechanism of evolution）**——但不是唯一机制（还有遗传漂变等）
- **竞争（Competition）**：因资源（食物、配偶、空间等）有限，个体之间必然竞争
- **差异生存（Differential Survival）**：拥有更有利表型的个体更可能生存
- **差异繁殖（Differential Reproduction）**：生存下来的个体产生更多后代，将有利性状传递下去

---

### LO 7.1.B：解释自然选择如何影响种群

**【进化适应度（Evolutionary Fitness）】**
- ⚠️ **适应度 ≠ 体能强壮**
- **进化适应度 = 繁殖成功（reproductive success）**
  - 即：一个个体相对于其他个体，对下一代基因库贡献的**基因比例**
  - 即使一个个体很强壮，如果它没有后代，其适应度为零

**【环境波动的影响】**
- **生物环境（Biotic）**：捕食者、竞争者、寄生物、食物来源等
- **非生物环境（Abiotic）**：温度、降水、光照、土壤成分等
- 这些环境因素会波动 → 影响进化的**速率和方向**
- 每一代可能选择**不同的遗传变异**——今天有利的性状，明天可能不利

> ⚠️ **考纲特别警示**：
> 1. 学生必须避免**拉马克式描述**（如"生物因需要而进化出某种特征"）
> 2. 慎用"fitness"等术语——必须解释其背后的**繁殖成功**概念
> 3. 要清楚：**自然选择作用于个体，但进化的是种群**

---

## 🌿 TOPIC 7.2 — Natural Selection（自然选择）

### LO 7.2.A：描述种群中表型变异的重要性

**【自然选择作用于表型变异】**

- **前提条件**：种群中必须存在**可遗传的表型变异**
- 自然选择**不能**创造新变异，它只能对**已有的变异**进行筛选
- 环境变化产生**选择压力（selective pressure）** → 某些表型被选择

**【选择类型（考纲要求理解）】**

| 选择类型 | 描述 | 对种群的影响 |
|----------|------|-------------|
| **定向选择（Directional）** | 选择一端的极端表型 | 种群均值向一个方向移动 |
| **稳定化选择（Stabilizing）** | 选择中间表型，淘汰两端极端 | 遗传多样性减少，种群保持稳定 |
| **分化选择（Disruptive）** | 选择两端极端表型，淘汰中间 | 可能导致**物种形成** |

---

### LO 7.2.B：解释细胞内分子的变异如何与生物体的适应度相联系

- 同一物种的不同个体之间，**细胞内分子的数量和类型**存在差异
- 这些分子层面的变异决定了生物体在特定环境中的生存和繁殖能力
- 分子多样性越高的种群 → 应对环境变化的能力越强

**💡 考纲示例**：
- **开花时间与全球气候变暖**：某些植物因气候变暖而提前开花 → 如果与其传粉昆虫的活动时间不同步 → 繁殖成功率下降
- **DDT抗性在昆虫中的进化**：在施用DDT前，少数昆虫携带抗性基因 → DDT杀死了不抗药的个体 → 抗药个体大量繁殖 → 种群整体产生抗药性
- **镰状细胞贫血（Sickle Cell Anemia）**：
  - 纯合子（Hb^S Hb^S）：严重贫血，适应度低
  - 纯合子（Hb^A Hb^A）：正常，但在疟疾流行区易感染疟疾
  - **杂合子（Hb^A Hb^S）**：轻微贫血症状，但对疟疾有**部分抵抗力** → **在疟疾流行区适应度最高** ← 杂合子优势

---

## 🌿 TOPIC 7.3 — Artificial Selection（人工选择）

### LO 7.3.A：解释人类如何影响种群内的多样性

**【人工选择 = 人类指导的进化】**

- 人类有意识地选择具有**理想性状**的个体进行繁殖
- 逐代积累 → 性状发生显著变化
- **与自然选择的区别**：选择压力来自**人类需求**而非自然条件

**【经典例子】**

| 领域 | 示例 | 选择结果 |
|------|------|---------|
| **农业** | 玉米从野生玉米（teosinte）驯化 | 籽粒变大、排列整齐、产量增加 |
| **畜牧业** | 狗的驯化 | 从狼到数百种不同品种（体型、毛色、性格各异） |
| **园艺** | 卷心菜品种选育 | 同一物种（Brassica oleracea）产生卷心菜、西兰花、花椰菜、羽衣甘蓝等 |

- ⚠️ 人工选择证明**自然选择是可行的**——如果人类能在短时间内改变一个物种，自然在更长时间内能做得更多

---

## 🌿 TOPIC 7.4 — Population Genetics（种群遗传学）

### LO 7.4.A：解释随机事件如何影响种群的遗传组成

**【驱动进化的随机过程】**

---

**① 突变（Mutation）**

| 特征 | 描述 |
|------|------|
| **随机性** | 突变不受生物体"需要"的指导——不是环境压力"定向产生"的 |
| **方向** | 可以是中性的、有害的或有益的（取决于环境） |
| **对进化的贡献** | **为种群添加新的遗传变异**——自然选择的原料来源 |

---

**② 遗传漂变（Genetic Drift）**

- **定义**：等位基因频率的**随机波动**，在**小种群**中特别显著
- **机制**：纯粹由"抽样误差"引起 → 某些等位基因因偶然事件而消失或固定
- **效果**：降低遗传多样性；使种群间产生**随机分化**

| 类型 | 描述 | 后果 |
|------|------|------|
| **瓶颈效应（Bottleneck Effect）** | 灾难性事件（火山爆发、火灾、过度捕猎）使种群**骤减至很少个体** | 幸存者的等位基因频率不能代表原种群 → 遗传多样性**急剧丧失** |
| **奠基者效应（Founder Effect）** | 少量个体从原种群**分离出去**，建立新种群 | 新种群的等位基因频率**不同于**源种群 → 某些稀有等位基因可能在新种群中高频出现 |

- **经典案例**：北象海豹（Northern elephant seal）曾被猎杀至约20头 → 现在虽恢复到数十万头，但**遗传多样性极低**（瓶颈效应）

---

**③ 基因流（Gene Flow）/ 迁移（Migration）**

- 个体迁入或迁出 → 增加或移除等位基因
- **效果**：
  - 增加迁入种群的遗传多样性
  - **减少种群间的遗传差异** → 阻止种群分化成不同物种
  - 使不同种群的基因库更加相似

---

### LO 7.4.B：描述随机过程在特定种群进化中的作用

- 突变 → 产生新的遗传变异 → 为自然选择提供"原材料"
- 遗传漂变 → 小种群可随机偏离同种的其他种群 → 可能导致分化
- 基因流 → 连接不同种群 → 阻止分化

### LO 7.4.C：描述种群遗传组成随时间的变化

- **等位基因频率的变化** = 进化发生的**直接证据**
- 只要 $p$ 或 $q$ 在代际间发生改变 → 进化已经发生

---

## 🌿 TOPIC 7.5 — Hardy–Weinberg Equilibrium（哈迪-温伯格平衡）

### LO 7.5.A：描述等位基因和基因型频率在种群中变化的条件

**【哈迪-温伯格平衡的定义】**
一个用于描述和预测**非进化种群**中等位基因和基因型频率的**数学模型**。

**【五个条件（必须同时满足）】**

| 编号 | 条件 | 违反后的后果 |
|------|------|-------------|
| ① | **大种群规模（Large population size）** | 遗传漂变将起作用 |
| ② | **无迁移（No migration / gene flow）** | 基因流会改变等位基因频率 |
| ③ | **无新突变（No new mutations）** | 新等位基因被引入 |
| ④ | **随机交配（Random mating）** | 非随机交配改变基因型频率 |
| ⑤ | **无自然选择（No natural selection）** | 某些等位基因被优先传递 |

> ⚠️ **这些条件在自然界永远不会被完全满足**！所以进化是必然的。
> **H-W平衡的真正价值**：提供一个**零假设（null hypothesis）**——如果观测到的基因型频率显著偏离H-W预期 → 证明进化正在发生！

---

**【核心方程式及用法】**

**方程1（等位基因频率）**：
$$p + q = 1$$

- $p$ = 显性等位基因的频率
- $q$ = 隐性等位基因的频率

**方程2（基因型频率）**：
$$p^2 + 2pq + q^2 = 1$$

- $p^2$ = 纯合显性的频率
- $2pq$ = 杂合子的频率
- $q^2$ = 纯合隐性的频率

**【标准解题步骤】**

1. 如果知道**隐性表型频率**（= $q^2$）→ 开方得 $q$
2. $p = 1 - q$
3. 计算 $p^2$ 和 $2pq$
4. 用**卡方检验**比较观测值与预期值 → 判断是否显著偏离H-W平衡

---

## 🌿 TOPIC 7.6 — Evidence of Evolution（进化的证据）

### LO 7.6.A：描述提供进化证据的数据类型

进化得到**多学科交叉证据**的有力支持：

| 学科 | 证据类型 | 示例 |
|------|---------|------|
| **地理学** | 生物地理学（biogeography） | 岛屿特有物种、大陆漂移与物种分布 |
| **地质学** | 化石年代测定 | 地层层序、同位素定年 |
| **物理学** | 放射性同位素定年 | 碳-14、钾-氩定年法 |
| **生物化学** | DNA/蛋白质序列比较 | 序列相似性反映亲缘关系 |
| **数学** | 计算遗传距离、分子钟 | 估算分歧时间 |

---

### LO 7.6.B：解释形态学、生物化学和地质学数据如何提供生物随时间变化的证据

**【三大类证据的详细展开】**

---

**① 化石证据（Fossil Evidence）**

- **化石形成**：生物遗骸被沉积物覆盖 → 矿物取代有机物质 → 石化
- **定年方法（3种）**：
  1. **岩石的相对年龄**：根据地质层序（越深层的岩石越古老）
  2. **放射性同位素定年**：利用已知半衰期（如碳-14半衰期5730年）
  3. **地理数据**：板块构造、古地理重建
- **过渡化石（Transitional fossils）**：展示中间形态
  - 例：始祖鸟（Archaeopteryx）——爬行动物到鸟类的过渡
  - 例：提塔利克鱼（Tiktaalik）——鱼类到四足动物的过渡

---

**② 形态学证据（Morphological Evidence）**

| 类型 | 定义 | 例证 |
|------|------|------|
| **同源结构（Homologous Structures）** | 不同生物中**源于共同祖先**的相似结构（但功能可能不同） | 人的手臂、猫的前腿、鲸的鳍、蝙蝠的翼——骨骼排列方式相同 |
| **痕迹结构（Vestigial Structures）** | 在进化中**失去原始功能**但仍保留的结构 | 人的阑尾、尾骨；鲸的后肢骨；蛇的骨盆残迹 |
| **同功结构（Analogous Structures）** | 功能相似但**起源不同**（趋同进化） | 鸟的翅膀与昆虫的翅膀——不是同源证据 |

> ⚠️ 考纲明确要求：**同源结构 ≠ 同功结构**。只有同源结构提供共同祖先的证据。

---

**③ 分子证据（Molecular Evidence）**

- **DNA序列比较**：两个物种共有的核苷酸序列相似度越高 → 亲缘关系越近
- **蛋白质序列比较**：如细胞色素c、血红蛋白等保守蛋白的氨基酸序列差异
- **分子钟（Molecular Clock）**：基于DNA序列的累积突变速率来估算物种分歧时间

> ⚠️ 考纲强调：**分子数据通常比形态学特征更准确可靠**，因为形态可能受趋同进化的影响。

---

## 🌿 TOPIC 7.7 — Common Ancestry（共同祖先）

### LO 7.7.A：描述细胞和分子水平上为所有真核生物共同祖先提供证据的结构和功能证据

**【所有真核生物共有的特征（来自共同祖先）】**

| 特征 | 描述 |
|------|------|
| **膜结合细胞器（Membrane-bound Organelles）** | 细胞核、线粒体、叶绿体（后两者仅存在于某些真核生物）等 |
| **线性染色体（Linear Chromosomes）** | 不同于原核生物的环状染色体 |
| **含有内含子的基因** | 基因被非编码内含子打断（真核生物基因组的普遍特征） |

- 这些特征在**所有真核生物**中出现 → 最合理的解释：它们继承了同一共同祖先的遗传蓝图

**【LUCA（Last Universal Common Ancestor）】**

- 所有现存生物（真核生物、细菌、古菌）的**最后一个普遍共同祖先**
- 证据：通用遗传密码、类似的代谢途径、核糖体结构的保守性等

## 🌿 TOPIC 7.8 — Continuing Evolution（持续进化）

### LO 7.8.A：解释进化是所有生物体中的持续过程

> ⚠️ **进化不是只在过去发生的——它正在我们周围持续进行！**

**【持续进化的证据（考纲举例）】**

| 例证 | 描述 |
|------|------|
| **基因组随时间的变化** | 对比不同年代的DNA序列（如古DNA与现代DNA）→ 基因组在不断变化 |
| **化石记录的持续变化** | 更厚的沉积层中记录的生物形态变化 |
| **抗生素耐药性** | 细菌对多种抗生素产生耐药性（如MRSA、耐碳青霉烯肠杆菌） |
| **农药/除草剂抗性** | 昆虫对DDT等杀虫剂的抗性；杂草对草甘膦的抗性 |
| **化疗药物耐药性** | 癌细胞在化疗压力下进化出耐药机制 |
| **新发传染病（Emergent Diseases）** | 病原体持续进化，跨越物种屏障 → COVID-19、禽流感等 |

## 🌿 TOPIC 7.9 — Phylogeny（系统发育）

### LO 7.9.A：描述可用于推断进化关系的数据类型

**【系统发育树 vs 分支图】**

| 特征 | 系统发育树（Phylogenetic Tree） | 分支图（Cladogram） |
|------|-------------------------------|-------------------|
| **是否显示时间尺度** | **是**（分支长度反映进化时间或变化量） | **否** |
| **是否显示进化差异量** | **是**（分支长度可表示遗传距离） | **否** |
| **是否可校准** | 可通过化石或分子钟校准 | 不可校准 |
| **本质** | 定量模型 | 定性模型 |

**【术语解释】**

| 术语 | 定义 |
|------|------|
| **节点（Node）** | 代表**最近共同祖先**（most recent common ancestor） |
| **分支（Branch）** | 代表一个谱系（lineage） |
| **外群（Out-group）** | 与其余生物**亲缘关系最远**的谱系，用于确定进化方向 |
| **共有的衍生特征（Shared Derived Characters）** | 存在于多个后代谱系中的衍生特征，指示共同祖先 |
| **共有原始特征（Shared Ancestral Characters）** | 从共同祖先继承的古老特征，不能区分亲缘关系 |

**【构建依据】**

1. **形态学相似性**：现存和化石物种的形态比较
2. **分子数据**：DNA序列、蛋白质序列
   - ⚠️ **分子数据通常比形态学特征更准确可靠**（因为趋同进化可能导致形态相似但不亲缘）

---

### LO 7.9.B：解释如何使用系统发育树和分支图推断进化亲缘关系

- 系统发育树和分支图是**假设（hypotheses）**，**不断根据新证据被修订**
- 两个物种在树上**共享节点越多** → 亲缘关系越近 → 从共同祖先分化后的时间**越短**
- 可用于说明**物种形成**的时间和模式

---

## 🌿 TOPIC 7.10 — Speciation（物种形成）

### LO 7.10.A：描述新物种可能出现的条件

**【物种形成的必要条件】**
- 两个种群之间出现**生殖隔离（Reproductive Isolation）**
- 基因流被阻断 → 种群独立进化 → 遗传差异积累到不能再交配的程度

**【生物学物种概念（Biological Species Concept）】**

- **定义**：物种 = 能够**相互交配并产生可育后代**的自然群体
- **局限性**：
  - 不适用于**无性生殖**生物（如细菌）
  - 不适用于**化石**（无法知道交配情况）
  - 不适用于**地理隔离**的群体（无法测试）

---

### LO 7.10.B：描述不同生态条件下进化和物种形成的速率

**【进化的时间模式】**

| 模式 | 描述 | 证据 |
|------|------|------|
| **渐变论（Gradualism）** | 进化通过微小的逐步变化，在**数百万年**内缓慢积累 | 某些化石记录显示渐进的形态变化 |
| **间断平衡（Punctuated Equilibrium）** | 物种在大部分时间处于**停滞（stasis）**，然后**在短时间内快速进化** | 化石记录中常见"突然"出现新物种，然后长期不变 |

> ⚠️ 两种模式**不互斥**，不同物种可能遵循不同模式。

---

**【进化的方向模式】**

| 模式 | 描述 | 示例 |
|------|------|------|
| **趋异进化（Divergent Evolution）** | 对**不同栖息地**的适应导致表型多样化 | 达尔文雀鸟喙形的多样化 |
| **适应性辐射（Adaptive Radiation）** | 新栖息地可用时，一个祖先进化出多个物种，**物种形成速率特别快** | 夏威夷果蝇、加拉帕戈斯群岛的达尔文雀 |
| **趋同进化（Convergent Evolution）** | **相似选择压力**导致**不相关**的种群/物种产生**相似表型适应** | 鲨鱼（软骨鱼）和海豚（哺乳动物）的流线体型 |

---

### LO 7.10.C：解释驱动物种形成的过程和机制

**【地理模式】**

| 类型 | 描述 | 机制 |
|------|------|------|
| **异域物种形成（Allopatric Speciation）** | 种群被**物理地理屏障**隔离（山脉、河流、海洋） | 地理隔离 → 基因流中断 → 独立进化 → 生殖隔离 |
| **同域物种形成（Sympatric Speciation）** | 种群的分布**有地理重叠** | 生态位分化、多倍体化（植物常见）、性选择等 |

---

**【生殖隔离机制（Reproductive Isolating Mechanisms）】**

**合子前隔离（Pre-zygotic）——阻止受精发生：**

| 机制 | 描述 | 例证 |
|------|------|------|
| **栖息地隔离** | 同一区域的物种占用不同栖息地，不接触 | 两种蛙——一种在池塘繁殖，一种在溪流繁殖 |
| **时间隔离** | 繁殖时间不同 | 两种植物—一种春季开花，一种夏季开花 |
| **行为隔离** | 求偶信号不同，不被识别 | 不同种萤火虫的闪光模式不同 |
| **机械隔离** | 生殖器结构不匹配 | 不同种蜗牛的生殖器朝向不同 |
| **配子隔离** | 配子在分子层面不能融合 | 精子和卵子的表面蛋白不匹配 |

**合子后隔离（Post-zygotic）——受精后出现问题：**

| 机制 | 描述 | 例证 |
|------|------|------|
| **杂种不可活** | 杂合子无法存活到性成熟 | 山羊×绵羊的杂种胚胎早期死亡 |
| **杂种不育** | 杂合子存活但不育 | 马×驴→**骡（mule）**：强壮但不育 |
| **杂种衰败** | 杂种F2代或回交后代适应性低下 | 某些植物杂种的第二代生长不良 |

---

## 🌿 TOPIC 7.11 — Variations in Populations（种群变异）

### LO 7.11.A：解释物种或种群的遗传多样性如何影响其抵御环境压力的能力

**【遗传多样性的进化重要性】**

- **影响种群动态**：变异水平直接决定种群应对环境变化的能力
- **低多样性 → 高风险**：遗传多样性低的物种面临**衰退或灭绝**的更大风险
  - 原因：缺乏足够多样性来应对新的选择压力
  - 近亲繁殖（inbreeding） → 有害隐性等位基因纯合 → 适应度下降
- **高多样性 → 高韧性**：更可能包含能**抵御环境压力**的个体

**【环境依赖性的等位基因适应度】**
- 在一个环境中**有利**的等位基因，在另一个环境中可能**有害**
- 例：镰状细胞等位基因在疟疾流行区有利，在非疟疾区有害

**💡 考纲举例**：
- **加利福尼亚秃鹫（California condor）**：曾仅剩20余只，近亲繁殖导致遗传多样性极低
- **黑足鼬（Black-footed ferret）**：几乎灭绝，现存个体源于极少数祖先
- **草原鸡（Prairie chicken）**：伊利诺伊州种群因多样性低导致繁殖成功率下降
- **马铃薯枯萎病（Potato blight）**：单一作物种植 → 遗传均一 → 病原体大量传播（爱尔兰大饥荒）
- **玉米锈病（Corn rust）**：类似的单一栽培问题
- **细菌的抗生素耐药性**：不是所有个体都对抗生素敏感 → 多样性高的种群更可能含有耐药个体

## 🌿 TOPIC 7.12 — Origins of Life on Earth（地球生命起源）

### LO 7.12.A：描述支持地球生命起源模型的科学证据

**【地质时间框架】**

| 事件 | 时间 | 说明 |
|------|------|------|
| **地球形成** | ~**46亿年前（bya）** | |
| **环境可支持生命** | ~**39亿年前** | 此前环境过于恶劣（频繁的陨石撞击、高温等） |
| **最早化石证据** | ~**35亿年前** | 叠层石（stromatolites）——微生物席形成的化石 |
| **生命起源窗口** | **39亿年 - 35亿年前** | 约4亿年的窗口期 |

> ⚠️ **进化 ≠ 生命起源**：进化解释生命多样性如何在出现后变化，但**不解释生命最初如何起源**。两者是不同的科学问题。

---

**【RNA世界假说（RNA World Hypothesis）】**

这是目前最广泛接受的生命起源模型。

**三个核心假设：**

1. **RNA作为最早的遗传物质**
   - 在DNA和蛋白质出现之前，RNA既携带遗传信息又具有催化功能
   - RNA可以自我复制（虽然效率低）——保障**遗传连续性**

2. **碱基配对是复制所必需的**
   - RNA链之间可以通过碱基互补配对进行模板指导的合成
   - 这是最简单的信息复制方式

3. **基因编码的蛋白质不参与催化**
   - 在最早的生命形式中，**核酶（Ribozymes，具催化功能的RNA）**承担了后来由蛋白质酶完成的功能
   - 蛋白质酶是后来进化出的"更高效的版本"

**【支持RNA世界假说的证据】**
- 核糖体中的rRNA是**肽酰转移酶（peptidyl transferase）**——催化肽键形成，是一种核酶
- RNA可以在实验室条件下从简单前体（核苷酸）非生物合成
- Miller-Urey实验证明：模拟早期地球条件下，有机分子（包括氨基酸、核苷酸碱基）可以自发形成

## 🌍 TOPIC 8.1 — Responses to the Environment（对环境响应）

### LO 8.1.A：解释生物的行为和生理响应如何与内部或外部环境变化相关

- 生物通过**行为机制**和**生理机制**响应环境变化
- **内部变化**：体温、血糖、激素水平等
- **外部线索**：光照、温度、湿度、捕食者、食物可用性等
- 生物之间通过交换信息来响应内部变化和外部线索 → 改变行为

**💡 举例（行为响应）**：
- **光周期现象（Photoperiodism）**：植物根据日照长度调节开花时间
- **向光性（Phototropism）**：植物向光生长
- **趋性（Taxis）**：动物**定向**运动朝向或远离刺激源（如飞蛾趋光）
- **动性（Kinesis）**：动物**非定向**活动速率随刺激强度变化（如潮虫在干燥环境中活动更快）
- **夜行性/昼行性（Nocturnal/Diurnal）**：根据光照周期调整活动节律
- **战斗或逃跑反应（Fight-or-Flight）**：动物面对威胁时的生理响应（肾上腺素释放等）

**💡 举例（信息交换）**：
- **捕食者警告信号**（如鸟类的警报叫声）
- **植物对植食的防御响应**（释放挥发性化学物质，吸引捕食黄蜂）

> ❌ **Exclusion**：特定行为和生理机制的详细知识不考。

---

### LO 8.1.B：解释生物的行为响应如何影响其整体适应度并可能促进种群的成功

**【生物通信方式（考纲明确列出5种）】**

| 通信方式 | 描述 | 例证 |
|----------|------|------|
| **视觉信号** | 通过颜色、动作、姿态传递信息 | 孔雀开屏、蜜蜂舞蹈 |
| **听觉信号** | 通过声音传递信息 | 鸟鸣、狼嚎、青蛙叫声 |
| **触觉信号** | 通过身体接触传递信息 | 蜜蜂的触角交流、灵长类的理毛行为 |
| **电信号** | 通过电场传递信息 | 某些鱼类（电鳗）使用电信号 |
| **化学信号** | 通过信息素等化学物质传递信息 | 蚂蚁的踪迹信息素、哺乳动物的领地标记 |

**【行为与适应度的关系】**

- **信号行为**可导致**差异繁殖成功（differential reproductive success）**
  - 更有效的求偶信号 → 更多的交配机会 → 更多的后代
  - 更有效的领地防御 → 更多的资源 → 更高的繁殖成功率
- **对信息的响应和信息通信**对**自然选择和进化至关重要**

**【先天行为 vs 习得行为】**
- **先天行为（Innate Behavior）**：天生具有，不依赖经验（如蜘蛛织网、海龟回归出生地）
- **习得行为（Learned Behavior）**：通过学习获得（如鸟类学会辨认捕食者）

**适应度偏向**那些能增加生存和繁殖成功的行为（无论先天还是习得）。

---

**【合作行为（Cooperative Behavior）】**

- 增加**个体适应度**和**种群生存**
- **看似"利他"的行为**，从进化角度看通常也增加个体的**包容适应度（inclusive fitness）**
- **亲缘选择（Kin Selection）**：通过帮助近亲繁殖，间接增加自身基因的传递

**💡 举例**：
- **亲代-子代互动**：亲代照顾 → 子代存活率提高
- **求偶和交配行为**：确保找到配偶并成功繁殖
- **蜜蜂和其他动物的觅食**：社会性昆虫分工合作
- **群体行为**：狼群（pack）、牛群（herd）、鸟群（flock）、鱼群（schooling）——降低被捕食风险
- **昆虫的群体和集群行为**：蚂蚁的社会、蜜蜂的舞蹈沟通
- **亲缘选择**：工蜂不育，但帮助母后繁殖（通过保护亲姐妹传递自己的基因）

> ❌ **Exclusion**：各种通信和行为系统的详细机制不考。

## 🌍 TOPIC 8.2 — Energy Flow Through Ecosystems（生态系统能量流）

### LO 8.2.A：描述生物获取和使用能量的策略

**【能量的基本用途】**
- 生物使用能量进行：**组织维持、生长、繁殖、稳态维持**

---

**【内温动物 vs 外温动物】**

| 特征 | 内温动物（Endotherms） | 外温动物（Ectotherms） |
|------|----------------------|----------------------|
| **体温来源** | 主要通过**代谢产热**（内部） | 主要来自**外部环境**（阳光、地面等） |
| **温度调控效率** | 高（有恒温机制） | 低（体温随环境波动） |
| **能量需求** | **高**（需要大量食物维持体温） | **低**（不需要代谢产热） |
| **举例** | 哺乳动物、鸟类 | 爬行动物、两栖动物、鱼类、昆虫 |
| 行为调节 | 较少依赖（主要通过生理调节） | 常通过行为调节（晒太阳、躲阴凉、聚集） |

---

**【能量平衡与生物健康】**

| 状态 | 结果 |
|------|------|
| **能量净增（Net Gain）** | 能量储存（如脂肪）→ 生长 → 繁殖输出增加 |
| **能量净损（Net Loss）** | 质量减少 → 繁殖输出下降 → 最终死亡 |

**【能量可用性与繁殖策略】**
- 不同生物根据**能量可用性**采用各种繁殖策略
- 有些生物在**无性繁殖和有性繁殖之间交替**：
  - 例：水蚤（Daphnia）在有利环境无性繁殖，环境恶化时转有性繁殖
  - 例：酵母在营养充足时出芽繁殖（无性），营养不足时有性繁殖

**💡 考纲示例**：
- **季节繁殖**：许多动物和植物在能量最充足的季节繁殖（如春季）
- **生活史策略**：两年生植物（第一年储存能量，第二年开花结果）、繁殖滞育（reproductive diapause，条件不利时暂停繁殖）

---

### LO 8.2.B：解释能量如何通过营养级流动以及物质如何循环

**【生态组织层次（由小到大）】**

```
个体（Individual）→ 种群（Population）→ 群落（Community）
→ 生态系统（Ecosystem）→ 生物群系（Biome）→ 生物圈（Biosphere）
```

---

**【能量流 vs 物质循环——核心区别】**

| 维度 | 能量流（Energy Flow） | 物质循环（Matter Cycling） |
|------|---------------------|-------------------------|
| **方向** | **单向**（太阳→生产者→消费者→热能散失） | **循环**（在环境和生物间周而复始） |
| **是否可循环** | **不可循环**——最终全部以热能形式散失 | **可循环**——物质守恒，不断再利用 |
| **驱动** | 持续需要**外部能量输入**（主要是太阳） | 由能量流驱动 |

---

**【三大生物地球化学循环的详细描述】**

---

**① 水循环（Hydrologic / Water Cycle）**

- **水库**：海洋（最大）、地表水（湖泊、河流）、地下水、大气（水蒸气）、冰川、生物体
- **过程**：
  - **蒸发（Evaporation）**：从海洋/地表水 → 水蒸气进入大气
  - **蒸腾（Transpiration）**：植物叶片的气孔释放水蒸气
  - **凝结（Condensation）**：水蒸气冷却 → 云
  - **降水（Precipitation）**：雨、雪、冰雹等
  - **径流（Runoff）** + 渗透（Infiltration）：水返回海洋和地下水

---

**② 碳循环（Carbon Cycle）**

- **关键反应**：
  - **光合作用（Photosynthesis）**：CO₂ + H₂O → 有机物（碳水化合物）+ O₂
  - **细胞呼吸（Cellular Respiration）**：有机物 + O₂ → CO₂ + H₂O + 能量
  - **分解（Decomposition）**：微生物分解有机体 → 释放 CO₂
  - **燃烧（Combustion）**：化石燃料燃烧 → 释放 CO₂（人类活动的关键影响）
- **碳库（Reservoirs）**：大气（CO₂）、化石燃料、海洋（溶解碳酸盐）、生物质（有机物）、岩石（碳酸盐岩）
- ⚠️ **人类活动通过燃烧化石燃料大量增加大气CO₂ → 全球气候变化**

---

**③ 氮循环（Nitrogen Cycle）——步骤最多、最复杂的循环**

| 步骤 | 过程 | 转化的物质 | 参与的微生物 |
|------|------|-----------|-------------|
| **① 固氮（Nitrogen Fixation）** | 将大气中的N₂转化为氨（NH₃） | N₂ → NH₃ | 固氮菌（如**根瘤菌**Rhizobium，与豆科植物共生） |
| **② 同化（Assimilation）** | 植物吸收NH₄⁺或NO₃⁻，用于合成蛋白质/核酸 | 无机氮→有机氮 | 不需要微生物（植物自身） |
| **③ 氨化（Ammonification）** | 分解有机含氮废物→释放NH₄⁺ | 有机氮→NH₄⁺ | 分解者（细菌和真菌） |
| **④ 硝化（Nitrification）** | NH₄⁺→NO₂⁻→NO₃⁻ | 氨→亚硝酸盐→硝酸盐 | 硝化细菌（Nitrosomonas, Nitrobacter） |
| **⑤ 反硝化（Denitrification）** | NO₃⁻→N₂（返回大气） | 硝酸盐→氮气 | 反硝化细菌 |

- **最大氮库**：**大气（78% N₂）**——但大多数生物不能直接利用N₂！
- 关键限制：固氮速率常常限制生态系统生产力

---

**④ 磷循环（Phosphorus Cycle）——不同于其他循环的关键点：无气体阶段**

- **唯一没有大气成分的循环**
- **水库**：岩石和沉积物（磷酸盐矿物）
- **过程**：
  1. **风化（Weathering）**：岩石分解 → 释放PO₄³⁻到土壤和地下水
  2. **吸收（Uptake）**：生产者从土壤中吸收磷酸盐 → 纳入ATP、DNA、RNA、磷脂等
  3. **食物链传递**：消费者进食 → 磷在营养级间传递
  4. **返回土壤**：分解（decomposition）和排泄（excretion）
  5. **沉积**：部分磷酸盐随径流进入海洋 → 沉积到海底 → 地质时间尺度后可能再次抬升成岩
- **限制性因子**：在淡水生态系统中，磷通常是**最限制初级生产力的营养元素**（与氮并列）

---

### LO 8.2.C：解释能量可用性的变化如何影响种群、群落和生态系统

- **阳光变化** → 影响初级生产力 → 影响所有营养级
- 生产者（producers）**生物量或数量的变化** → 级联影响整个食物网

**【完整营养级结构】**

```
第4营养级：    四级消费者（Quaternary Consumers）— 顶级捕食者
                    ↑
第3营养级：    三级消费者（Tertiary Consumers）
                    ↑
第2营养级：    次级消费者（Secondary Consumers）
                    ↑
第1营养级：    初级消费者（Primary Consumers）= 草食动物
                    ↑
第0营养级：    生产者（Producers）= 自养生物
                    ↑
              （太阳/无机化学能）
```

- **分解者（Decomposers）**：在所有营养级工作，分解死亡有机物，释放养分
- **能量传递效率**：约 **10%** 从上一营养级传递到下一营养级（其余以热能散失）
  - 这解释了为什么**营养级通常不超过4-5级**

---

### LO 8.2.D：解释自养生物和异养生物的活动如何实现生态系统内的能量流动

**【自养生物（Autotrophs）】**

可以从**无机来源**捕获能量的生物。

| 类型 | 能量来源 | 碳源 | 例证 |
|------|---------|------|------|
| **光合自养（Photoautotrophs）** | **太阳光** | CO₂ | 植物、藻类、蓝细菌 |
| **化能自养（Chemoautotrophs）** | **无机化学反应**（如H₂S、NH₃、Fe²⁺） | CO₂ | 深海热泉的硫细菌、硝化细菌 |

- **光合自养生物** → **初级生产力（Primary Productivity）**的基础
- **化能自养生物** → 支持深海热泉等**无光生态系统**

---

**【异养生物（Heterotrophs）】**

必须从**有机物质**中获取能量的生物。

| 类型 | 食物来源 | 例证 |
|------|---------|------|
| **草食动物（Herbivores）** | 直接吃生产者 | 牛、鹿、毛虫 |
| **肉食动物（Carnivores）** | 吃其他消费者 | 狮子、鹰、蛇 |
| **杂食动物（Omnivores）** | 植物和动物都吃 | 人类、熊、乌鸦 |
| **分解者（Decomposers）** | 分解死亡有机物 | 细菌、真菌 |
| **食腐动物（Scavengers）** | 吃已死生物 | 秃鹫、蚯蚓 |

- 异养生物**代谢碳水化合物、脂质和蛋白质**作为能量来源
- 他们通过**消耗**来自自养生物有机物质来捕获能量

---

## 🌍 TOPIC 8.3 — Population Ecology（种群生态学）

### LO 8.3.A：描述影响种群增长动态的因素

**【种群（Population）的定义】**
- 同一物种的个体，在**同一时间、同一区域**内，彼此相互作用并与环境相互作用

**【种群增长动力学】**

种群增长取决于三个基本参数：
- **出生率（Birth Rate, B）**：单位时间内产生的新个体数
- **死亡率（Death Rate, D）**：单位时间内死亡的个体数
- **当前种群大小（Population Size, N）**

**【种群增长方程】**

$$\frac{dN}{dt} = B - D$$

其中 $\frac{dN}{dt}$ = 种群规模随时间的变化率

**【指数增长（Exponential Growth）】**

- 条件：资源**不受限制**，繁殖**无约束**
- J形曲线
- 公式：$\frac{dN}{dt} = r_{max}N$
- $r_{max}$ = **最大人均增长率（Maximum per capita growth rate）**
- 现实中只在短时间内或特定条件下实现（如细菌在培养皿中、入侵物种在新环境中）

---

## 🌍 TOPIC 8.4 — Effect of Density on Populations（密度对种群的影响）

### LO 8.4.A：解释种群密度如何受环境中资源可用性的影响和决定

**【环境容纳量（Carrying Capacity, K）】**

- **定义**：生态系统在现有资源条件下能长期支持的**最大可持续种群规模**
- **不是固定不变**——随环境条件和资源可用性变化

**【密度制约因素（Density-dependent factors）】**

| 因素 | 机制 | 效果随密度增加 |
|------|------|--------------|
| **竞争（Competition）** | 个体争夺有限资源（食物、空间、配偶） | **加剧** |
| **捕食（Predation）** | 捕食者集中在种群密度高的区域 | 捕食压力**增加** |
| **疾病（Disease）** | 高密度促进病原体传播 | 传播速率**增加** |
| **废物积累** | 自身代谢废物（如氨）积累 | 毒性效应**增加** |

**【非密度制约因素（Density-independent factors）】**
- 影响**与种群密度无关**
- 例证：自然灾害（火灾、洪水、飓风）、极端天气、火山喷发、污染事件

**【逻辑斯蒂增长（Logistic Growth）】**

- 条件：资源有限 → 增长受限制
- S形曲线
- 公式：$\frac{dN}{dt} = r_{max}N\frac{K-N}{K}$
- 当 $N$ 远小于 $K$：$(K-N)/K \approx 1$ → 接近指数增长
- 当 $N$ 接近 $K$：$(K-N)/K \approx 0$ → 增长率趋近于0
- 当 $N$ 超过 $K$：$(K-N)/K$ 为负 → 种群下降

## 🌍 TOPIC 8.5 — Community Ecology（群落生态学）

### LO 8.5.A：根据物种组成和多样性描述群落结构

- **群落（Community）** = 同一区域中**所有不同物种的种群**的集合
- **物种组成**：群落中包含哪些物种
- **物种多样性**：包含**物种丰富度（richness，物种数）** 和**均匀度（evenness，各物种个体数的分布）**

**【辛普森多样性指数（Simpson's Diversity Index）】**

$$Diversity\ Index = 1 - \sum\left(\frac{n}{N}\right)^2$$

- $n$ = 某一特定物种的个体总数
- $N$ = 所有物种的总个体数

**解释**：
- 指数值范围：0 ~ 1
- **值越接近1** → 多样性越高（物种丰富度高，均匀度好）
- **值越接近0** → 多样性越低（可能只有少数物种占绝对优势）

---

### LO 8.5.B：解释种群内部和种群之间的相互作用如何影响群落结构

**【种间相互作用类型】**

| 相互作用 | 物种1 | 物种2 | 描述 |
|----------|-------|-------|------|
| **捕食（Predation）** | + | – | 一种生物（捕食者）吃掉另一种生物（猎物） |
| **竞争（Competition）** | – | – | 两种生物争夺相同资源 |
| **互利共生（Mutualism）** | + | + | 双方受益 |
| **偏利共生（Commensalism）** | + | 0 | 一方受益，另一方不受影响 |
| **寄生（Parasitism）** | + | – | 一种生物（寄生物）生活在另一种生物（宿主）上或体内，获取营养 |
| **合作（Cooperation）** | + | + | 同种个体间的互利行为 |

---

**【高级群落生态学概念（考纲要求）】**

| 概念 | 定义 |
|------|------|
| **捕食者-猎物动态（Predator-Prey Dynamics）** | 捕食者和猎物种群数量相互震荡的经典模式（Lotka-Volterra模型） |
| **营养级联（Trophic Cascade）** | 顶级捕食者通过控制中间捕食者，间接影响较低营养级 |
| **生态位分化（Niche Partitioning）** | 竞争导致物种在资源利用上产生差异，减少直接竞争 |
| **关键种（Keystone Species）** | 对群落结构影响**不成比例**大的物种（即使数量不多） |
| **生态演替（Ecological Succession）** | 群落组成随时间的变化（初生演替/次生演替） |

## 🌍 TOPIC 8.6 — Biodiversity（生物多样性）

### LO 8.6.A：描述生态系统多样性与其对环境变化韧性之间的关系

**【多样性-稳定性假说（Diversity-Stability Hypothesis）】**

- **组分少、多样性低的生态系统** → **韧性较低** → 对环境变化更脆弱
  - 例：单一作物田（monoculture farm）→ 一种病害即可毁灭全部作物
- **组分多、多样性高的生态系统** → **韧性较高** → 能更好承受和恢复从干扰
  - 例：热带雨林 → 丰富的物种形成复杂的生态网络，有"功能冗余"

---

### LO 8.6.B：解释生态系统组分的添加或移除如何影响其短期和长期结构

**【关键种（Keystone Species）的独特作用】**

- **定义**：对生态系统结构和功能的影响**远超其生物量/丰度所预期**的物种
- **特征**：关键种的移除 → 生态系统**可能崩溃**
- **经典案例**：
  - **海獭（Sea otter）**：捕食海胆 → 保护海藻林 → 维持整个生态系统
  - **狼（Gray wolf）**：捕食麋鹿 → 防止过度啃食 → 维持植被和溪流生态
  - **蜜蜂（Pollinators）**：传粉 → 支撑大量开花植物的繁殖

**【关键种移除的效果链】**

```
关键种移除
    ↓
被捕食物种数量暴增
    ↓
过度消耗资源
    ↓
其他物种失去栖息地/食物
    ↓
生态系统崩溃或剧烈转变
```

## 🌍 TOPIC 8.7 — Disruptions in Ecosystems（生态系统扰动）

### LO 8.7.A：解释环境与种群中随机或预先存在的变异之间的相互作用

**【适应（Adaptation）的定义】**
- 被**自然选择偏好的遗传变异**，表现为在特定环境中为生物提供**优势**的性状
- ⚠️ **适应不是"为了"某个目的而产生的**——它只是因提高适应度而被选择保留
- ⚠️ **突变不"定向"产生**：突变是随机的，环境不能"指导"生物产生特定的突变

**【杂合子优势（Heterozygote Advantage）】**

- 杂合子比任一纯合子具有**更高的相对适应度**
- **经典案例：镰状细胞贫血**
  - Hb^A Hb^A（正常）：在疟疾流行区易感染疟疾
  - Hb^S Hb^S（镰状）：严重贫血，适应度低
  - **Hb^A Hb^S（杂合）**：轻微贫血但**对疟疾有抵抗力** → 在疟疾流行区适应度最高

---

### LO 8.7.B：解释入侵物种如何影响生态系统动态

**【入侵物种（Invasive Species）的特征】**

| 特征 | 描述 |
|------|------|
| **引入方式** | **有意或无意**（通过人类活动引入非本地区域） |
| **竞争优势** | 在新环境中缺少**天敌、竞争者和寄生物** → 不必将能量用于防御 |
| **繁殖能力** | 通常繁殖快、产生大量后代 |
| **生态位利用** | 可能占据**空余生态位**或**直接排挤本地物种** |
| **后果** | 降低本地生物多样性、改变生态系统结构、造成经济损失 |

**💡 经典案例**：

| 入侵物种 | 原产地 | 引入地 | 影响 |
|----------|--------|--------|------|
| **葛藤（Kudzu）** | 日本/中国 | 美国东南部 | 疯狂蔓延，覆盖本地植被 |
| **斑马贻贝（Zebra mussels）** | 里海 | 北美五大湖 | 堵塞管道，排挤本地贻贝 |
| **野猪（Feral pigs）** | 欧亚 | 全球 | 破坏植被和土壤 |
| **缅甸蟒（Burmese python）** | 东南亚 | 佛罗里达沼泽地 | 几乎消灭了当地小型哺乳动物 |

---

### LO 8.7.C：描述导致生态系统结构和动态变化的人类活动

**【人类加速的环境变化】**

| 活动/现象 | 描述 |
|-----------|------|
| **生物放大作用（Biomagnification）** | 有毒物质（如DDT、汞）在食物链中**逐级富集** → 顶级捕食者体内浓度最高 |
| **富营养化（Eutrophication）** | 过量氮磷（来自农业化肥、污水）进入水体 → **藻类爆发** → 藻类死亡后分解耗氧 → 水体**缺氧** → 鱼类死亡 |
| **全球气候变化（Global Climate Change）** | 燃烧化石燃料 → CO₂增加 → 温室效应→ 全球变暖、极端天气 |
| **伐木（Logging）** | 森林破坏 → 生物多样性丧失、碳汇减少、水土流失 |
| **城市化（Urbanization）** | 栖息地破碎化、岛屿化 → 种群孤立 → 遗传多样性丧失 |
| **单一作物种植（Monocropping）** | 遗传均一性 → 易受病虫害爆发（如爱尔兰马铃薯饥荒） |
| **过度捕捞/狩猎** | 直接导致物种濒危或灭绝 |

---

**【自然扰动（考纲也列举）】**

| 现象 | 影响 |
|------|------|
| **厄尔尼诺（El Niño）** | 太平洋海面温度异常 → 全球气候异常 → 影响生态系统 |
| **大陆漂移（Continental Drift）** | 地质时间尺度上改变物种分布和隔离模式 |
| **陨石撞击（Meteor Impact）** | 白垩纪末（6500万年前）陨石撞击 → 恐龙灭绝 → 为哺乳动物腾出生态位 |
